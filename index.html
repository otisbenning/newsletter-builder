📚 Newsletter Builder Anleitung

🚀 Wie nutze ich den Editor?

  1. Module hinzufügen: Ziehen Sie Module aus der linken Sidebar in den Vorschau-Bereich
  2. Bearbeiten: Klicken Sie auf "✎ Bearbeiten" um Inhalte anzupassen
  3. Reihenfolge ändern: Nutzen Sie ↑↓ oder ziehen Sie Module per Drag & Drop
  4. Events importieren: Klicken Sie im Events-Modul auf "📥 Events importieren"

📥 Events vom Portal importieren

  1. Ziehen Sie das Bookmarklet in Ihre Lesezeichen-Leiste
  2. Öffnen Sie portal.startwithafriend.de/events
  3. Klicken Sie auf das Bookmarklet → Events werden in Zwischenablage kopiert
  4. Fügen Sie die Events im Editor ein (Strg+V)

📤 Newsletter in CleverReach einfügen

  1. Klicken Sie auf "📋 HTML kopieren" (oben rechts im Editor)
  2. Gehen Sie zu CleverReach → Neuen Newsletter erstellen
  3. Wählen Sie "HTML-Editor" als Template-Typ
  4. Fügen Sie den kopierten HTML-Code ein (Strg+V)
  5. Speichern und Vorschau prüfen

💾 Vorlagen speichern

  1. Speichern: Klicken Sie auf "Speichern" → Name eingeben → Im Browser gespeichert
  2. Laden: Klicken Sie auf "Laden" → Vorlage auswählen
  3. Export JSON: Als Datei exportieren (zur Weitergabe/Backup)
  4. Import JSON: JSON-Datei importieren

💡 Tipps

  • Pflicht-Module (Preheader, Header, Footer) werden automatisch sortiert
  • Nutzen Sie {FIRSTNAME[std:Hi]} für Personalisierung in CleverReach
  • Markieren Sie wichtige Events als "⭐ Highlight"
  • Testen Sie den Newsletter auf Mobile (F12 → Handy-Symbol)
